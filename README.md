# movie-web-resource-archive
该仓库主要用于收集和归档影视相关网站的公开基础资料，涵盖电影、电视剧、综艺、动漫等不同类型的影视站点。整理内容仅限站点层级的客观信息，不包含任何影视内容文件、播放链接或资源引用。所有资料均整理自公开渠道，目的在于形成稳定、可持续更新的站点信息档案，减少重复检索成本。本项目不参与站点运营，不对站点内容质量、版权属性或更新频率作出判断，仅作为中性的信息整理与资料归档用途存在。
